# 🌤 天气预报小助手

**作者：JP** ✨

一个超级可爱又实用的微信天气插件～  
输入城市名 + “天气”，马上就能收到美美哒实时天气报告啦！☀️🌧️❄️

### 使用方法 💕
- 在聊天中发送：**城市名 + 天气**  
  示例：  
  `广州天气`  
  `北京天气`  
  `上海天气`

- 配置指令：**天气配置**  
  发送这个指令就能打开设置界面，配置 Token 和白名单哦～

### 如何获取彩云天气 Token 🔑
1. 打开这个链接注册/登录账号：  
   https://platform.caiyunapp.com/login
2. 登录后在控制台找到你的 **Token**（一串长长的字符串）
3. 在插件里打开“天气配置” → 粘贴 Token → 保存 🌟

配置好后，所有开启白名单的聊天就能愉快地查天气啦～
